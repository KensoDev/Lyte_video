Lyte_video by bainternet

Lyte_video is a jquery plugin to embed youtube videos on selected event, after the page loads. This is a very small jQuery plugin (under 1kb minified gziped) that will let you "late load" youtube videos and save requests untill the user actualy clicks (or any other event) the play on the video and only then the video is embbeded.

How To Use?

1. Include nessesary JS files

            <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.7/jquery.min.js"></script>
            <script type="text/javascript" src="/fancybox/lyte_video.jQuery.min.js"></script>

2. create an element with v_id attribute and set the value to a video it

          <div id="content">
          <div class="lz_embed" v_id="MKIxhq8elrU"></div><br/><br/>
          <div class="lz_embed" v_id="y-SpsT1dIJ0"></div><br/><br/>
          <div class="lz_embed" v_id="YxGlfiP-3UA"></div><br/><br/>
          </div>

3. last thing is to call the plugin on the container element

          <script type="text/javascript">
            jQuery(document).ready(function(){
              jQuery('#content').lyte_video();
            });
          </script>

Options and API

You can pass options as key/value object to lyte_video() function

OptionDefualtDescription
event'click'event to bind
selector'lz_embed'jQuery class selector
width'560'video width
height'315'video height
suggestedfalseShow suggested videos
httpsfalseuse HTTPS
privacyfalseEnable privacy mode
embed_oldfalseUse old embed code
play_hdfalsePlay In HD
lang'en_US'youtube player language (not sure is this works)
thumbnail0Show video thumbnail (values: 0, 1, 2, 3, custom image URL, false to invoke manually)
